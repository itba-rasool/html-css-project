Project Overview

🎯

Goal

Build a static webpage using only HTML and CSS. No JavaScript, no frameworks — just clean fundamentals.

📋

Requirements

  • Proper semantic HTML
  • Headings, sections & images
  • Clean, readable layout
  • External CSS only
⚡

Skills Gained

  • HTML5 fundamentals
  • CSS Box Model
  • Flexbox & Grid
  • Accessibility (A11Y)

Key Concepts

HTML

Semantic Structure

Use <header>, <main>, <footer>, <article> so browsers and screen readers understand your page.

CSS

Separation of Concerns

All styles go in one external CSS file. No inline styles — one source of truth is easier to maintain.

LAYOUT

Grid & Flexbox

CSS Grid for the overall page structure. Flexbox for aligning elements inside components.

A11Y

Accessibility

Alt text on all images. One <h1> per page. Color contrast ratio above 4.5:1 — always.
